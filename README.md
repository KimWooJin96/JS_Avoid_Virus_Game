# Simple Game - Avoid Virus

Using HTML, CSS, JavaScript

## Game info

- Win : Save all people by clicking them
- Lose :
  - Click any virus
  - Couldn't save everyone for the rest of the time
- Pause : Pause the game and start new

## Skills

- HTML/CSS
- Vanilla JS
